# Project instructions

This is an R package project.

Please follow these rules:
- Do not make large rewrites unless requested.
- Preserve my existing function names and coding style.
- Use roxygen2 documentation style.
- Prefer small, targeted fixes.
- Before changing exported functions, explain the reason.
- For package checks, use:
  Rscript -e "devtools::check()"
- For documentation, use:
  Rscript -e "devtools::document()"
- For tests, use:
  Rscript -e "devtools::test()"
